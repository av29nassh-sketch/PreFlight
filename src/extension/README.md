# PreFlight Companion (Beta)

Real-time AI code drift and security guarding for the PreFlight Pro Engine.

## Quick Start

This extension requires the PreFlight Pro CLI daemon to be running locally on your machine.

```bash
npm install -g preflight-pro
preflight start
```

## Features

**Real-time Background Scanning:** Instantly detects structural code changes and security anomalies via local AST parsing as you type.

**Automated Bridge Setup (Path B):** Seamlessly spawns an interactive setup terminal directly inside VS Code if the core engine daemon isn't detected.

**Instant Security Alerts:** Keeps your database security and application architecture tightly guarded against unverified AI agent modifications.
